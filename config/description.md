This Nextstrain build is a minor adaptation of the [Nextstrain seasonal influenza build](https://github.com/nextstrain/seasonal-flu) to minimize sampling and maximize inclusion of user samples in the build.

Links for references used: [H3N2 HA](https://github.com/INSaFLU/seasonal-flu/blob/master/config/h3n2/ha/reference.fasta), [H1N1 HA](https://github.com/INSaFLU/seasonal-flu/blob/master/config/h1n1pdm/ha/reference.fasta), [B/Vic HA](https://github.com/INSaFLU/seasonal-flu/blob/master/config/vic/ha/reference.fasta).

Please cite the original Nextstrain work if you use results from this build.

This adaptation work had the support of the [DURABLE project](https://durableproject.org).
